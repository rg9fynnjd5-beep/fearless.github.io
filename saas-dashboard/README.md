# SaaS Dashboard

This is a modern dark-themed SaaS dashboard scaffold built with React, Vite, Tailwind CSS, Framer Motion, and Firebase.

Quick start:

1. Install dependencies

```bash
cd saas-dashboard
npm install
```

2. Start dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Deploy: set Netlify build command to `npm run build` and publish directory `dist`.

Replace Firebase values in `src/firebase.js` with your project config.
